# Retail Data Analytics Using SQL
## Overview

This project uses SQL to analyze structured data and extract actionable business insights. It covers data cleaning, exploratory data analysis (EDA), and analytical querying to understand patterns in sales, customers, and products. The analysis is performed using MySQL in MySQL Workbench.

## Objective

1. Analyze retail sales transactions to identify customer purchasing patterns

2. Segment customers based on total quantity purchased

3. Identify repeat buyers and loyal customers

4. Support data-driven marketing, inventory, and retention strategies

## Customer Segmentation Strategy

Customers are categorized based on total quantity of products purchased:

1. No Orders – 0 purchases

2. Low – 1 to 10 purchases

3. Mid – 11 to 30 purchases

4. High Value – More than 30 purchases

Datasets are included with the project.
