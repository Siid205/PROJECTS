# Retail Data Analytics Using SQL
Overview

This data analytics project focuses on analyzing retail business data using SQL to uncover insights related to customer purchasing behavior, customer segmentation, and sales performance. The project applies structured query techniques to transform raw transactional data into meaningful business intelligence that supports marketing, inventory, and customer retention strategies.

The initiative demonstrates an end-to-end analytics workflow using SQL, covering data cleaning, aggregation, segmentation logic, and behavioral analysis. The project highlights practical SQL skills used in real-world retail analytics scenarios.

Project Scope

The project follows a systematic analytics approach consisting of the following stages:

Explore and prepare the data: Data cleaning and exploratory analysis using SQL

Build analytical queries: Aggregations, joins, and CASE-based segmentation

Analyze customer behavior: Repeat purchases, loyalty, and spending patterns

Generate business insights: Actionable findings for decision-making

Key Goals

Analyze retail sales transactions to identify customer purchasing patterns

Segment customers based on total quantity purchased

Identify repeat buyers and loyal customers

Support data-driven marketing, inventory, and retention strategies

Customer Segmentation Strategy

Customers are categorized based on total quantity of products purchased:

No Orders – 0 purchases

Low – 1 to 10 purchases

Mid – 11 to 30 purchases

High Value – More than 30 purchases
